# Logo concepts — Andrew Malone Design

Six candidate marks to replace the current peak-and-wave-in-a-circle logo.
All six are drawn in the site's warm ink (`#1C1A17` from `src/styles/tokens.css`),
work at 16 px favicon size, and ship as standalone SVGs. Swap the fill/stroke to
`currentColor` (or `#F6F4EE`) for use on dark grounds.

| # | File | Concept |
|---|------|---------|
| 01 | `01-ridgeline.svg` | Closest evolution of the current mark: peak (A) over wave (m), no enclosing circle, two round strokes. |
| 02 | `02-tile.svg` | Favicon-first: the peak knocked out of a solid rounded square. Carries its own ground, so it never dissolves against a tab bar. The knockout is a true mask, not a painted line. |
| 03 | `03-assembly.svg` | Initials as a geometry kit — a triangle for the A, a dome for the m, on a shared baseline. The most abstract option. |
| 04 | `04-fold.svg` | One strip folded into a peak; the lighter return face (55 % opacity of the same ink) gives depth without a second color. |
| 05 | `05-am.svg` | Lowercase typographic mark, `am.`, drawn as geometry rather than typed in a font. Doubles as a case-study sign-off. |
| 06 | `06-daybreak.svg` | A solid sun crossed by the original wave: A.M. — literally morning. Keeps the circular equity of the old logo, flipped from outline to solid. |
